# TikTokデモ動画素材

FlowDoChatのTikTok動画制作用に作成された動的デモHTMLファイル群です。

## 🎬 シナリオ: 「疲れた朝の一歩」

**コンセプト:** 親身なAIパートナーが、疲れた気持ちに寄り添いながら、小さな一歩を一緒に踏み出す

| シーン | ユーザー | AIの対応 |
|--------|---------|----------|
| 1. 傾聴 | 「疲れたよぉ...今日もう何もしたくない」 | 「お疲れ様...！🫂 そういう日、あるよね」 |
| 2. 提案 | 「うん...でもなんかやらなきゃって思う」 | 「えらい！超簡単なことから始めよう！」 |
| 3. 実行 | 「じゃあ水飲もうかな」 | 「いいね！🥤 やってみよう！」 |
| 4. 称賛 | 「飲んだ！」 | 「えらい！！🎉 ほら、一歩できたじゃん！」 |

---

## 📁 ファイル構成

| ファイル | 用途 | 使い方 |
|---------|------|--------|
| **ChatDemo_自動再生.html** | 画面録画で一発撮り | 「再生」ボタンで自動アニメーション |
| **ChatDemo_シーン切替.html** | 個別シーンの確認 | ボタン or 矢印キーで切り替え |
| **AI動画生成用_スチル.html** | Pika/Runway/Sora用 | 4フレームを個別キャプチャ |

---

## 🛠️ 使い方

### 1. 画面録画 (ChatDemo_自動再生.html)

```
1. ブラウザでファイルを開く
2. 「再生」ボタンをクリック
3. macOSの画面録画 (Cmd + Shift + 5) で録画開始
4. 動画編集ソフトでトリミング
```

### 2. シーン切替 (ChatDemo_シーン切替.html)

| 操作 | 動作 |
|------|------|
| ボタンクリック | シーン切り替え |
| `←` / `→` キー | 前後シーンへ移動 |
| `A` キー | 3秒ごとの自動再生ON/OFF |

### 3. AI動画生成ツール用 (AI動画生成用_スチル.html)

```
1. ブラウザを100%ズームで表示
2. 各フレームを Cmd + Shift + 4 で個別キャプチャ
3. Pika / Runway / Sora などにアップロード
```

**推奨プロンプト例:**
```
Smooth transition between frames, mobile app UI animation, 
chat bubbles appearing with gentle motion, warm and friendly feel,
modern iOS design with glassmorphism effects
```

## 🎨 デザイン仕様

- **ベース:** `ChatPatern.html` から抽出
- **比率:** 9:16 (TikTok縦動画対応)
- **iPhoneフレーム:** 375x812px (iPhone X相当)
- **カラーパレット:** Emerald〜Teal グラデーション

### AIアバターの色分け
| 色 | 役割 | モード |
|----|------|--------|
| 🟡 Amber | ケア (Care) | 傾聴 |
| 🔵 Sky Blue | ルーティン (Routine) | 提案 |
| 🟣 Purple | フォーカス (Focus) | 実行・称賛 |
